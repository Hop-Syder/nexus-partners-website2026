### Structure du Projet

```
nexus-partners.xyz/
├── src/
│   ├── app/
│   │   ├── [locale]/                     # Routes internationalisées (fr / en)
│   │   │   ├── (site)/
│   │   │   │   ├── about/                # Page À Propos + layout SEO
│   │   │   │   ├── services/             # Page Services + layout SEO
│   │   │   │   ├── work/                 # Page Portfolio
│   │   │   │   ├── contact/              # Page Contact + layout SEO
│   │   │   │   ├── privacy-policy/
│   │   │   │   ├── terms-and-conditions/
│   │   │   │   └── legal-notices/
│   │   │   ├── layout.tsx                # Layout racine + metadata globales
│   │   │   └── page.tsx                  # Page d'accueil
│   │   ├── api/                          # Routes API
│   │   ├── components/                   # Composants UI réutilisables
│   │   ├── robots.ts                     # Configuration robots.txt
│   │   └── sitemap.ts                    # Sitemap XML dynamique
│   ├── i18n/routing.ts                   # Routing i18n
│   └── proxy.ts                          # Proxy Next.js 16 (i18n)
├── messages/
│   ├── fr.json                           # Traductions françaises
│   └── en.json                           # Traductions anglaises
└── public/                               # Assets statiques
```

### SEO & Indexation

| Élément                                   | Statut                                           |
| ----------------------------------------- | ------------------------------------------------ |
| `robots.txt`                              | ✅ Tous crawlers autorisés                       |
| Agents IA (GPTBot, Claude, Perplexity...) | ✅ Explicitement whitelistés                     |
| `sitemap.xml`                             | ✅ Généré dynamiquement (FR + EN)                |
| Balises `hreflang`                        | ✅ Sur toutes les pages                          |
| `openGraph`                               | ✅ Complet sur toutes les pages                  |
| Directives `googleBot`                    | ✅ `max-snippet: -1`, `max-image-preview: large` |
| JSON-LD Schema.org                        | ✅ `Organization` structuré                      |

## 📦 Installation & Développement

### Prérequis

- **Node.js** 18+
- **npm** ou yarn

### Commandes

```bash
# Installer les dépendances
npm install

# Serveur de développement (Turbopack)
npm run dev

# Build de production
npm run build

# Démarrer en production
npm run start

# Analyse ESLint
npm run lint
```
